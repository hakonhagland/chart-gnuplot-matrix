# Chart::Gnuplot::Matrix

Plot matrix using Gnuplot

## Installation

```
perl Makefile.PL
make
make install  # (or run with sudo if you are using the system perl)
```

alternatively you can run `cpanm .` from the folder with
`Makefile.PL`.

## Test installation

- Go to folder `examples/` and run `p.pl`:

```
cd examples
p.pl
```
